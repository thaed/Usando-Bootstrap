Bienvenido a Bootstrap

Bootstrap es un marco de diseño (framework) de código abierto para diseñar sitios web y aplicaciones móviles.

Bootstrap te permite crear sitios web de manera más rápida y fácil gracias a sus componentes predefinidos y su sistema de rejilla (grid).
